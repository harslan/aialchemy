# Agent AIChemy — The Intent Defense Challenge

Registration page for the April 27, 2026 event at Suffolk University.

**Live:** https://harslan.github.io/aialchemy/

## Before the event

Replace `REPLACE_WITH_FORM_ID` in `index.html` with a Formspree form ID:

1. Sign up at https://formspree.io (free tier handles 50 submissions/month)
2. Create a new form; copy the form ID (e.g. `xyzabc123`)
3. In `index.html`, find `https://formspree.io/f/REPLACE_WITH_FORM_ID` and paste the ID in place of the placeholder
4. Commit and push — Pages redeploys automatically

Until then, the form blocks submission with an alert so nothing is lost silently.

## Deploy

GitHub Pages is set to serve from the `main` branch, root. Any push to `main` redeploys.
